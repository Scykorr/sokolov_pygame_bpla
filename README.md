# Quad-Copter vs Missile Simulation

Симуляция боевого взаимодействия между квадрокоптером и ракетой на pygame.

## Требования

- Python 3.13+
- pygame 2.4.0+

## Установка

1. **Активировать виртуальное окружение:**
   ```bash
   .\venv\Scripts\activate
   ```

2. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

```bash
python main.py
```

или с пользовательскими параметрами:

```bash
python main.py <скорость_дрона> <скорость_ракеты> <радиус_зоны>
```

### Примеры:
```bash
# Быстрый дрон
python main.py 300 200 3

# Медленный дрон
python main.py 50 150 2

# Большая зона
python main.py 100 200 5
```

**По умолчанию** (если параметры не указаны):
- Скорость дрона: 100 км/ч
- Скорость ракеты: 200 км/ч
- Радиус зоны: 3 км

Закрыть программу можно, нажав крестик на окне.

## Как работает симуляция

- **Синий квадрат** - квадрокоптер (в центре)
- **Красный треугольник** - ракета
- **Зелёная окружность** - зона обнаружения/преследования квадрокоптера
- **Жёлтый круг** - взрыв при перехвате

Квадрокоптер автоматически начинает двигаться внутри зоны к ракете при её входе.
Соответственно, если дрон достаточно быстр, он не даст ракете выйти из зоны и перехватит её.

## Проект использует

- **pygame** - для отрисовки и обработки событий
- **Python 3.13** - как базовая версия интерпретатора
