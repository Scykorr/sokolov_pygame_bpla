# Quad-Copter vs Missile Simulation

Интерактивная симуляция боевого взаимодействия между квадрокоптером и ракетой с GUI на pygame.

## Требования

- Python 3.13+
- pygame 2.4.0+
- Pillow 10.0.0+ (для загрузки спрайтов)

## Установка

1. **Активировать виртуальное окружение:**
   ```bash
   .\venv\Scripts\activate
   ```

2. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Генерирование спрайтов (опционально):**
   ```bash
   python generate_sprites.py
   ```
   Если спрайты не найдены, программа будет использовать геометрические фигуры.

## Запуск

```bash
python main.py
```

## Интерфейс программы

### Экран настройки
При запуске программы вам будут доступны три поля ввода:
1. **Скорость дрона (км/ч)** - скорость квадрокоптера
2. **Скорость ракеты (км/ч)** - скорость угрозы
3. **Радиус зоны (км)** - зона обнаружения/преследования дрона

Значения проверяются на корректность. Нажмите **НАЧАТЬ СИМУЛЯЦИЮ**, чтобы запустить.

### Экран симуляции
Во время работы симуляции доступны кнопки:
- **ПАУЗА** - приостановить/возобновить симуляцию
- **СБРОСИТЬ** - вернуться к экрану настройки

### Элементы симуляции

**Роль объектов:**
- **Синий объект (дрон)** - атакующий квадрокоптер, появляется с края экрана, движется в направлении ракеты
- **Красный объект (ракета)** - защитная ракета, неподвижна в центре, отслеживает и перехватывает дрон
- **Зелёная окружность** - радиус зоны обнаружения дрона
- **Жёлтая линия** - линия преследования (видна когда ракета отслеживает дрон)
- **Жёлтый/красный круг** - эффект взрыва при успешном перехвате

**Визуализация:**
- Если найдены спрайты (drone.png и rocket.png), используются они (размер 40x40 px)
- Иначе используются геометрические фигуры (больший размер для лучшей видимости)
- Дрон вращается в соответствии с направлением движения

## Как работает симуляция

1. Ракета появляется на краю экрана и двигается в случайном направлении
2. Каждые 2-6 секунд ракета меняет направление примерно на ±30 градусов
3. Когда ракета входит в зелённую зону, квадрокоптер начинает двигаться ей навстречу
4. Если дрон достаточно быстр, он перехватит ракету до того, как она выйдет из зоны
